✍️ Work Monitor

A simple and interactive **Work Monitor web app** to manage daily tasks with priority levels, due dates, and persistent storage using localStorage.

## Features:
* Add tasks with date & time
* Set priority (High / Medium / Low)
* Mark tasks as completed
* Delete tasks
* Data saved using **localStorage** (persists after refresh)
* Clean and responsive UI with background styling

## Tech Stack:
* **HTML** – Structure
* **CSS** – Styling & UI design
* **JavaScript** – Functionality & logic
* **localStorage** – Data persistence

## 💡 How to Use
1. Enter a task
2. Select date & time
3. Choose priority
4. Click **Add Task**
5. Click on task to mark complete
6. Click ❌ to delete
